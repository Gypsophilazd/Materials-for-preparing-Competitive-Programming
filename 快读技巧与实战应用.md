#### 标准模板
```
inline int read() { 
	int x = 0, f = 1; 
	char ch = getchar(); 
	while(ch < '0' || ch > '9') { 
		if(ch == '-') f = -1; 
		ch = getchar(); 
	} 
	while(ch >= '0' && ch <= '9') { 
		x = (x << 1) + (x << 3) + (ch ^ 48); 
		ch = getchar(); 
	} 
	return x * f; 
}
```

#### (1) 符号构建模块:当读取字符不是数字时,只检测负号并一直读取(2)数值构建模块:如果读取是数字,利用位运算进行加速
